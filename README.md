# Cattpuccin-mocha Dark (customized)

omarchy-catppuccin-dark theme but a little customized with more backgrounds and very little personal things.

This theme uses the omarchy-catppuccin-dark [palette](https://catppuccin.com/palette/).

# Installation

To install this theme, simply use the omarchy-theme-install command:

```bash
omarchy-theme-install https://github.com/ABetancourt03/omarchy-catppuccin-dark-customized
```

or

`Omarchy menu > Install > Style > Theme` and use the following link:

`https://github.com/ABetancourt03/omarchy-catppuccin-dark
`

## Starship

For the same terminal look copy the `starship.toml` into your `~/.config` directory
